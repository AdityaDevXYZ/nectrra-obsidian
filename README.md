# Nectrra / Obsidian

**Sovereign Hyper-Sparse Swarm Intelligence**

Obsidian is a revolutionary, decentralized AI model designed to achieve 95%+ intelligence on global benchmarks without relying on capital-heavy datacenter clusters. It replaces raw financial capital with architectural ingenuity, harvesting idle computational cycles from standard consumer devices across a global peer-to-peer mesh.

## Core Architectural Pillars

1. **The Compute Network (Mesh)**: A custom distributed coordination protocol built on Libp2p to discover nodes and sync asynchronous states across firewalls.
2. **Mathematical Quantization**: 
   - *State Space Model (SSM)*: Linear time context ensuring O(1) inference complexity without quadratic RAM explosions.
   - *1-Bit Extreme Sparsity*: Ternary `{-1, 0, 1}` weights that completely bypass floating-point matrix multiplications, replacing them with hyper-fast additions executable instantly on mobile and edge CPUs.
3. **Multi-Tier Execution (Split-Brain Routing)**: A lightweight Local Reflex Engine handles immediate tasks, dynamically routing complex logic to the Global Reasoning Gateway (P2P swarm).
4. **Decentralized MCTS (System 2 Swarm Reasoning)**: The node spawns Monte Carlo Tree Search branches across the peer-to-peer swarm to solve complex logic puzzles collaboratively.
5. **Autonomous RLAIF (Golden Data Synthesis)**: Idle CPU cycles are utilized to autonomously synthesize code/math prompts, verify them in secure local sandboxes, and build pristine, mathematically flawless training corpora.
6. **Byzantine-Resilient Weight Aggregation**: A "Proof of Useful Training" filtering mechanism (Geometric L2 Norm Outlier Rejection) that instantly rejects poisoned gradient updates from malicious nodes in the permissionless swarm.

## Repository Structure

- `obsidian-node/`: The background orchestrator daemon combining the Local Reflex Engine and Global Reasoning Gateway.
- `obsidian-p2p/`: The Libp2p mesh networking crate for Swarm task routing and Federated gradient updates.
- `obsidian-ml/`: The inference crate featuring the custom Ternary math layer and State Space recurrence loops.
